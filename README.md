# 📦Automatic Box Folding Machine

---

## 🧩Tech Stack
- Design and Simulation: SolidWorks, AutoCAD, EPLAN.
- Control Hardware: Siemens S7-1200 (CPU 1214C DC/DC/DC).
- Software Programming: TIA Portal (Ladder Logic), WinCC (HMI).

---

## 🛠️Mechanical Design
- Developed comprehensive 3D models and complex mechanical assemblies using SolidWorks.
- Conducted motion simulations to ensure zero-interference and smooth kinematics between mechanical components.
- Designed custom mounts for pneumatic actuators and high-speed folding mechanisms.

## ⚙️Control System & Automation
- Programmed robust Ladder Logic (LAD) in TIA Portal to manage the precise folding sequence and timing.
- Implemented PROFINET communication for seamless data exchange between the PLC and industrial peripherals.
- Designed a user-friendly HMI/WinCC dashboard for real-time monitoring and manual system overrides.
---

## ⭐Key Achievements
- Mechanical Precision: Achieved perfect synchronization between pneumatic actuators and specific mechanical folding steps.
- Robust Control: Successfully implemented error-handling routines and emergency stop safety sequences for industrial reliability.
- System Integration: Built a fully functional and stable prototype from scratch, bridging mechanical hardware with intelligent control software.
